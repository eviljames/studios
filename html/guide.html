StudioS for Slackware 12.2 Build Guide


Build Environment:

The following guide has been built using Slackware 12.2+patches with the following series installed: A, AP, D, L, N, T, X, and XAP.


A Note of Thanks:

We would like to thank those involved with http://www.slackbuilds.org. If it was not for their hard work, this project would not be possible.


Audacity http://audacity.sourceforge.net


1. Install wxPython 2.8.9.2 (from SBo) http://slackbuilds.org/repository/12.2/libraries/wxPython/ (Dep of Audacity)


2. Install libsndfile 1.0.18 (from SBo) http://slackbuilds.org/repository/12.2/libraries/libsndfile/ (Dep of Audacity)


3. Install audacity 1.3.7 (from SBo) http://slackbuilds.org/repository/12.2/audio/audacity/
